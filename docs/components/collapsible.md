# collapsible

**Role:** `collapsible`  
**Fragment id:** `blocks.collapsible`  
**Twig:** `Collapsible` (+ `Collapsible:Trigger`, `Collapsible:Content`)

Single-region disclosure — one trigger and one content panel. Distinct from `accordion` (multi-item stack).
